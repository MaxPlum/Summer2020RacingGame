### Controller Input
<ul>
  <li>RT/LT for forwards and backwards.</li>
  <li>Right Analog Stick for turning left and right.</li>
  <li>Left Analog Stick for rotating camera.</li>
  <li>RB/LB for strafing left and right.</li>
  <li>A Button for jumping.</li>
</ul>
